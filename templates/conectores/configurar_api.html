Configurar API da Corretora (Binance)

⚠️ Defina no Render → Settings → Environment:
  • BINANCE_API_KEY = sua API Key
  • BINANCE_API_SECRET = seu API Secret
  • OPENAI_API_KEY = sua OpenAI API Key
Por segurança, recomendo usar variáveis de ambiente e não salvar chaves no banco.
